Android Kotlin Utility

WiFi Monitor

An Android application designed to monitor the device's WiFi connectivity state in real-time and alert the user via Toast notifications.

I developed this application to solve a personal pain point: incurring unexpected mobile data charges when my home router would turn off during power cuts. The app provides immediate feedback when the WiFi connection drops, allowing users to disable mobile data or take other actions immediately.
